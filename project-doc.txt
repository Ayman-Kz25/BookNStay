=> Client Side

1. React Setup with vite
npm create vite@latest
npm install
npm run dev

2. Tailwind CSS
npm install tailwindcss @tailwindcss/vite

3. React Router DOM
npm install react-router-dom

4. Lucide React Icons
npm install lucide-react

5. Clerk (For Authentication)
npm install @clerk/react

=> Server Side
1. setup Node Pkg Manager
npm init -y

2. Setup and Install Express Pkg
npm install express

3. Env Variables Pkg Setup
npm install dotenv

4.To Connect Backend with frontend URL
npm install cors

5. Pkg to connect backend with MongoDB
npm install mongoose

6. To upload the Imgs on Cloud Storage
npm install cloudinary

7. Helper Pkg to upload Img
npm install multer

8. To make API call using http request
npm install svix
